Abstract

Transformer-based Large Language Models (LLMs) have exhibited remarkable success in extensive tasks primarily attributed to the self- attention mechanism, which requires a token to consider all preceding tokens as its context to com- pute attention. However, when the context length L becomes very large (e.g., 128K), the amount of potentially redundant information in the con- text tends to increase. The redundant context not only hampers the modeling representation perfor- mance but also incurs unnecessary computational and storage overhead. In this paper, we propose a plug-and-play Core Context Aware (CCA) Atten- tion for efficient long-context modeling, compris- ing two complementary modules: 1) Globality- aware pooling module groups input tokens and dynamically compresses each group into one core token based on their significance. In this way, our method automatically focuses and strength- ens core context while diminishing redundancy during the learning process, leading to effective long-term dependency modeling. 2) Locality- preserving module incorporates neighboring to- kens to preserve local context for detailed repre- sentation. Notably, our CCA-Attention is able to replace the self-attention module in existing LLMs with minimal fine-tuning cost. Extensive experimental results show the superiority of our method in both long-context modeling and compu- tational efficiency over state-of-the-art methods.

Method Overview

Efficiency: Achieves linear complexity \(O(Lm + Ls)\) vs. quadratic \(O(L^2)\), 7.9x speedup on 128K context
Memory: Reduces KV cache by 93% on 128K context

Illustration of CCA-Attention, which includes two components: 1) Globality-aware pooling module encapsulates the input tokens X into core tokens C according to the importance (Eqn. (2)). The core tokens C serve as representative proxies of X for attention, thereby reducing computational costs. 2) Locality-preserving module incorporates the local context from neighboring tokens, acting as supplement for the globality-aware pooling module. We produce the final output Att by fusing these two modules based on Eqn. (5).

We propose a novel attention mechanism that replaces standard self-attention in Transformers to handle long contexts efficiently. CCA-Attention consists of two synergistic components: 1)Globality-aware Pooling Module and 2)Locality-preserving Module.


Globality-aware Pooling Module

Given input sequence $\mathbf{X} \in \mathbb{R}^{L \times d}$, we:

  1. Group tokens into $m = \lceil L/g \rceil$ groups, each with $g$ tokens
  2. Compute core tokens via importance-weighted pooling:
    \[ \mathbf{c}_i = \text{softmax}\left(\frac{\mathbf{Q}_{ig}\mathbf{K}^\top_{\mathcal{I}_i}}{\sqrt{d}}\right)\mathbf{X}^G_{\mathcal{I}_i} \]
    where $\mathcal{I}_i = \{(i-1)g+1, ..., ig\}$ is the $i$-th group's indices
  3. Derive global keys/values:
    \[ \widetilde{\mathbf{K}}^{\rm G}_{\mathcal{T}_i} = [\mathbf{K}^{\rm G}_1; \cdots; \mathbf{K}^{\rm G}_j], \quad \mathbf{K}^{\rm G} = \mathbf{C}\mathbf{W}^K \]
    where:
    • $\mathbf{C} = [\mathbf{c}_1; ...; \mathbf{c}_m]$ is the core token sequence
    • $j = \max(0, \lfloor(i-s)/g \rfloor)$ ensures causal attention
    • $\mathbf{W}^K$ are standard key projection parameters

Locality-preserving Module

Maintains a local window of \(s\) neighboring tokens to preserve fine-grained details:

\[ \widetilde{\mathbf{K}}^{\rm L}_{\mathcal{U}_i} = [\mathbf{K}^{\rm L}_k; \cdots; \mathbf{K}^{\rm L}_i] \]
\[ k = \max(1, i-s-((i-s) \bmod g)) \]
Fusion & Complexity

Combines both modules via differentiable fusion:

\[ \mathbf{Att}_i = \text{softmax}\left(\frac{\mathbf{Q}_i[\widetilde{\mathbf{K}}^{\rm G}_{\mathcal{T}_i}; \widetilde{\mathbf{K}}^{\rm L}_{\mathcal{U}_i}]^\top}{\sqrt{d}}\right)[\widetilde{\mathbf{V}}^{\rm G}_{\mathcal{T}_i}; \widetilde{\mathbf{V}}^{\rm L}_{\mathcal{U}_i}] \]

Experimental Results

Comprehensive Benchmark Evaluation

We evaluate CCA-LLM across multiple benchmarks to assess its effectiveness in long-context modeling. Our experiments compare against three categories of baselines:

  • Full self-attention (Vanilla SA): The standard Transformer baseline
  • Fixed-pattern sparse attention: StreamingLLM, LM-Infinite
  • Dynamic sparse attention: MInference

Comparisons on LongBench-E:

Table 1 demonstrates CCA-LLM's superior performance on the LongBench-E benchmark with LLaMA2-7B at 32K and 80K context lengths with a 3.5× speedup and 46% memory reduction compared to Vanilla SA.


Inference Flexibility:

CCA-Attention enables real-time adaptation to computational constraints by dynamically adjusting its attention granularity, achieving optimal speed-accuracy tradeoffs without model retraining.


Computational Efficiency:

CCA-LLM achieves end-to-end acceleration across all inference stages while significantly reducing memory footprint, outperforming existing methods (like MInference) that only optimize partial computation phases.

7.9x inference speedup on 128K tokens and 93% KV cache reduction in memory usage compared to Vanilla Self-Attention.

Key Insights

  1. Dynamic Compression Works: Core token compression effectively preserves semantic information while reducing computation
  2. Local Context is Crucial: The locality-preserving module accounts for 23% of performance gains in our ablation
  3. Practical Deployment: Tunable g/s parameters allow adaptation to different hardware constraints
